## Guidelines

### Do's
- Always review AI-generated code before committing
- Run tests on AI-generated changes
- Document AI-assisted modifications
- Verify security implications
- Maintain coding standards

### Don'ts
- Don't commit code without understanding it
- Don't skip security reviews
- Don't bypass testing procedures
- Don't ignore linting warnings
- Don't share sensitive information with AI tools
