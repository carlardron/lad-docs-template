# Documentation

This directory contains general notes, documentation, and reference materials.

## Organization

Use subdirectories to organize documents by project, topic, or category:

```
docs/
├── projects/
│   ├── project-alpha/
│   └── project-beta/
├── technologies/
│   ├── docker/
│   ├── kubernetes/
│   └── nodejs/
└── reference/
    ├── apis/
    └── configurations/
```

## Creating Documents

```bash
# Create a new note in this directory
npm run new
# Select: docs

# Create with subdirectories
npm run new
# Select: docs
# Subdirectories: projects/project-alpha
```

## Tips

- Use descriptive filenames
- Add tags to your documents for easy searching
- Link related documents together
- Keep a table of contents for large document sets
